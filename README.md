# gosabun

## ブラウザだけで完結するテキスト差分比較ツール

[デモ](https://gosabun.pages.dev) · [機能](#機能) · [使い方](#使い方) · [インストール](#インストール) · [カスタマイズ](#カスタマイズ) · [技術スタック](#技術スタック) · [開発](#開発)

---

> **完全クライアント完結** — 入力テキストはサーバーに送信されません。すべての処理はブラウザ内で完了します。

---

## 課題

2つのテキストの違いを確認したいだけなのに、ツールをインストールしたり、テキストを外部サーバーに送信したり、英語圏向けの diff ツールで日本語が正しく比較されなかったりする。

gosabun はシンプルな UI をベースに、**クライアント完結**・**日本語対応**・**カスタマイズ可能**なテキスト差分比較ツールを提供します。

---

## 動作イメージ

```text
┌──────────────┐       ┌──────────────┐
│ テキスト A    │  [⇄]  │ テキスト B    │
│              │       │              │
│ 探偵の田中は  │       │ 探偵の鈴木は  │
│ ダイヤモンドが│       │ エメラルドが  │
│ 消えた       │       │ 消えた       │
└──────────────┘       └──────────────┘

        [比較する ⌘+Enter]  [クリア]

  比較: [単語|1文字]  配色: [青|緑|モノ]
  表示: [全体|差分のみ]  ☑ 前後の空白を無視

  ─── 3 / 5 件  [▲ 前へ] [▼ 次へ] ───

   1 │ 探偵の[田中]は     │ 探偵の[鈴木]は
   2 │ [ダイヤモンド]が   │ [エメラルド]が
   3 │ 消えた             │ 消えた

  A: 24字 / 空白0 / 改行2 / 5語
  B: 22字 / 空白0 / 改行2 / 5語
```

2つのテキストを貼り付けて `Cmd+Enter`（または比較ボタン）を押すだけ。変更箇所がハイライト表示され、前後ナビゲーションで差分を順に確認できます。

---

## 機能

- **2つの比較モード** — 単語単位・1文字単位
- **3つのカラーテーマ** — 青・緑・モノ
- **表示フィルタ** — 全体 / 差分のみ
- **空白無視オプション** — 行頭・行末の空白を無視して比較
- **ダークモード** — ライト / ダーク切替対応
- **変更箇所ナビゲーション** — 前へ / 次へボタンで差分箇所にジャンプ
- **テキスト統計** — 文字数・空白数・改行数・単語数を表示
- **キーボードショートカット** — `Cmd+Enter` / `Ctrl+Enter` で即座に比較実行
- **レスポンシブ対応** — モバイルからデスクトップまで対応

## インストール

**前提条件**: Node.js 20 以上

```bash
git clone https://github.com/sidedishjob/gosabun.git
cd gosabun
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開くと使えます。

---

## 使い方

1. **テキスト A** と **テキスト B** にそれぞれテキストを入力（またはペースト）
2. **比較** ボタンをクリック、または `Cmd+Enter` / `Ctrl+Enter`
3. 差分がハイライト表示される
4. **▲ / ▼** ボタンで変更箇所を順に確認

テキストの入れ替え（⇄ボタン）やクリアも可能です。

## カスタマイズ

### 比較モード

| モード | 説明                               |
| ------ | ---------------------------------- |
| 単語   | 英単語をまとめて比較（デフォルト） |
| 1文字  | 1文字ずつ比較                      |

### カラーテーマ

| テーマ | 差分の色 |
| ------ | -------- |
| 青     | ブルー   |
| 緑     | グリーン |
| モノ   | グレー   |

### その他のオプション

- **表示モード**: 全体 / 差分のみ
- **空白無視**: 行頭・行末の空白を無視して比較
- **ダークモード**: 画面右のトグルで切替

| フレームワーク    | Next.js 16 + React 19   |

## Cloudflare Pages への CI/CD

本リポジトリでは、CI は GitHub Actions、CD は Cloudflare Pages の Git 連携で運用します。

- CI（`.github/workflows/ci.yml`）
  - `develop` / `main` 向け PR で `test` / `lint` / `format:check` / `build` を実行
  - `develop` への push でも同じ検証を実行
- CD（Cloudflare Pages 側）
  - Cloudflare Pages の GitHub 連携で `main` 更新時に自動デプロイ

### 事前準備

- Cloudflare Pages で GitHub リポジトリを連携
- Production branch を `main` に設定
- GitHub 側で `main` を保護（PR 必須 / 必要な CI チェック必須）すると、PR マージ経由のみ本番反映にできます
